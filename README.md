### Text file processing

Command | Objective
--------|----------
`sed '/pattern/d' <file>` | Delete lines containing `pattern` and print to `stdout`
`sed -i '' '/pattern/d' <file>` | Delete lines containing `pattern` and overwrite the `<file>` (works for BSD - Mac)
